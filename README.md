# 2D Planar Part Recognition

这是一个专为工业分拣比赛、机器人视觉抓取场景设计的 2D 平面零件材质识别系统。该项目针对工业场景中常见的**铝件 (aluminum)**、**锌合金 (zinc)** 和**电镀件 (plated)** 进行了深度的特征提取与算法优化，并提供了开箱即用的工业级预测 API 接口。

---

##  项目核心亮点

1. **多曝光融合 (Mertens 算法)**：针对金属零件容易出现反光、过曝或光照不足（欠曝）的问题，采用 OpenCV 的 Mertens 算法对多张不同曝光度的无损 BMP 图片进行无缝融合，还原真实表面细节。
2. **多维特征拼接 (Color + Texture)**：
   * **颜色特征 (HSV)**：提取色调、饱和度的均值与标准差。完美捕捉电镀件微弱倒影环境色彩（如麻布、木桌）的色调漂移。
   * **纹理特征 (LBP)**：通过局部二值模式 (Local Binary Pattern) 算子提取表面精细纹理直方图。精准区分铝件的拉丝划痕、锌合金的压铸颗粒感以及电镀件的极致光滑度。
3. **稳健分类器 (SVM)**：使用支持向量机（Support Vector Machine）进行特征分类，不仅具备优异的鲁棒性，还能实时计算分类结果的概率置信度，为后续机械臂决策（如低置信度阈值拦截）提供支持。

## 🚀快速上手指南

1. **环境依赖安装**：在运行代码前，请确保安装了以下 Python 依赖库：

``` Python
pip install opencv-python numpy scikit-image scikit-learn joblib
```
2. **图像采集**：在不同背景、曝光度、照度下拍摄图片。
   * **背景要求**：白纸、麻布等等无尖锐纹理的背景。
   * **照度要求**：从500-1000，每隔100拍摄一组。
   * **曝光要求**：从20000-120000，每隔20000拍摄一张。
   * **格式要求**：转换为bmp格式。

3. **预处理**：将拍摄好的图片按照目录结构放置，然后运行
``` Python
python pre_make.py
```

4. **模型训练**：运行
```
python train_model.py
```
，生成workpiece_svm_model.pkl模型文件

5. **最终输入**：将待测零件的图片放入根目录中，然后运行
```
python main.py [图片路径]
```
，程序会输出识别结果和置信度。
